Help

ClearViz simplifies the concept of natural language processing by providing an easy to use and modern interface.

Creating a Tree.
  1. Click link to create tree.
  2. Add title to tree. Content can be left blank if need be. (Content can be edited later)
  3. Click Submit.
  4. Analyzed tree should be shown.

Deleting a Tree.
  1. Click tree that you wish to delete.
  2. Delete link is below the title.
